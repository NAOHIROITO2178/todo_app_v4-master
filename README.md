# みんなのSchool Todo(Todoページ）
### これは、小学生から大学生に向けて開発したもので、合計８つのTodo管理リストが用意されている。カテゴリとしては、勉強・部活動・友達・恋愛・進路・学校行事・趣味・その他と分かれている。
